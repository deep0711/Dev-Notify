Welcome !!

How to use:

Firstly,pin the extension icon on Extension Menu.

Now,on clicking the extension icon, it provides a list of different platforms,from which you can check all those from which you want to be notified.

This extension gives notifications about all events(contest,hiring challenges,Hackathons,webinar,etc.) which are going to take place on your selected platforms in near future.

Each Notification consist of few details about event:
1.Name of Event
2.Start Date and Time of Event (in your Local time)
3.Platform Icon
4.Two buttons will be there:
(i)Take me there:Visit website to register
(ii)Don't show this contest:If you dont want to receive notification of that event again, click this button

Anytime you feel to update your preferences,just again check your required platforms and click Save

Notification Pattern:
1.As soon as you Update your preferences,the notifications will start
2.At every 3hrs ,new notifications will come(if you have saved your preferences)

Important:

For receiving Chrome Notifications,it must be allowed in system.
Check Here:
For Windows User : Settings -> System -> Notifications & actions
For Mac User : Apple Menu -> System Preferences -> Notifications
